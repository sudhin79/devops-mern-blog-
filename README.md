# devops-mern-blog

A full-stack blog app built with the MERN stack, containerised with Docker,
and deployed via a Jenkins CI/CD pipeline on AWS EC2.

## Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB (Docker container)
- CI/CD: Jenkins
- Monitoring: Prometheus + Grafana
- Cloud: AWS EC2

## Run locally with Docker
docker-compose up --build
